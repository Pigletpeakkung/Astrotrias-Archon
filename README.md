### **Netlify Deployment**
This portfolio is optimized for Netlify with:

```toml
# netlify.toml
[build]
  publish = "."
  command = "npm run build"

[[headers]]
  for = "/*"
  [headers.values]
    X-Frame-Options = "DENY"
    X-Content-Type-Options = "nosniff"
```
